# sequences/ — finalized C9 coding sequences
Per-species subfolders with the finalized C9 copy sequences: `<species>_<copy>.cds.fa` (CDS),
`.pep.fa` (protein), `.exons.fa` (per-exon), and `.PROVENANCE.txt` where relevant.
81 functional/kept copies (55 C9A + 22 C9B + 4 C9C), all intact ORFs (ATG + stop + 0 internal stops).
Corrected copies (Naja/Notechis C9B) and truncated-but-coding copies (Anolis sagrei C9A, Pantherophis
C9B, Anolis carolinensis C9A) are here; genuine pseudogenes/lost are in `../Pseudogenes/`.
Copy naming: C9A = DAB2-proximal, C9B = FYB1-proximal, C9C = colubrid third copy.
