# Anolis (iguanian) C9 duplication — ancestral copy & independence (2026-07-03)

Same 3 tests as the snake analysis, applied to the anole C9A/C9B duplication. Single-copy reference =
the 8 single-copy iguanians (Bradypodion, Cyclura, Furcifer, Gambelia, Laudakia, Phrynosoma, Pogona,
Sceloporus).

## Test 1 — synteny position
anole **C9A = DAB2-proximal** (both anoles), anole **C9B = FYB1-proximal**; single-copy iguanians
**8/8 DAB2-proximal**. => anole C9A holds the ancestral (DAB2-proximal) iguanian position.

## Test 2 — ML tree
The 4 anole copies + 8 single-copy iguanians form a **monophyletic iguanian clade with NO non-iguanian
leaves inside** => the anole C9A/C9B duplication is **INDEPENDENT of the snake duplication** (a separate,
iguanian-specific tandem event). Both anole copies are ~equidistant from snake C9A (0.82) and far from
snake C9B (0.94) — they are not orthologous to snake C9A/C9B. Single-copy iguanians cluster ~equally with
anole C9A (0.403) vs C9B (0.407).

## Test 3 — pairwise %identity
single-copy iguanian vs anole C9A = 77.0% ≈ anole C9B = 76.9% (**tied**). Anole paralog divergence
C9A<->C9B ~83% (carolinensis 82.9%, sagrei 84.3%); within-copy carol-vs-sagrei ~92-94% => duplication
**predates the carolinensis-sagrei split**. Anole copies ~66% to snake C9A and to snake C9B (equidistant
=> independent duplication).

## Conclusion
- The anole (iguanian) C9 duplication is **INDEPENDENT** of the snake one. anole "C9A" is NOT orthologous
  to snake "C9A" — the name reflects **positional homology** (both DAB2-proximal), not orthology.
- **anole C9A = ancestral-position copy** (DAB2-proximal, matches single-copy iguanians); **anole C9B =
  derived tandem duplicate** (FYB1-proximal). Duplication predates carolinensis/sagrei.
- CAVEAT: sequence tests (identity, tree distance) are **tied** for which anole copy is the ancestral
  lineage (deep duplication + likely C9A<->C9B gene conversion) — so the anole ancestral-copy call rests
  on **synteny position**, which is unambiguous (C9A ancestral, C9B derived).
