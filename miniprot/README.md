# miniprot/ — miniprot protein-to-genome C9 models
Per-species miniprot alignments of reference C9 proteins to each genome (locus-anchored gene models,
GFF + derived exon/CDS), used to finalize exon boundaries and reconcile chimeric/borrowed exons. The
locus-anchored `miniprot_corrected` models are the authoritative non-borrowed set (see METHODS §4).
