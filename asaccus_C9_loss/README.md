# asaccus_C9_loss/ — evidence for C9 gene loss in Asaccus caudivolvulus
`asaccus_C9_tblastn.outfmt6.tsv` = genome-wide tblastn of C9 protein vs the Asaccus genome (no intact
copy recovered); `Asaccus_caudivolvulus_pooledC9A_SRA.out.sam` = SRA pooled-read check; `asaccus_tblastn_
genomewide.sh` = the script. The assembly is chromosome-level (contig N50 ~20 Mb, 60x), so the absence is
genuine gene loss, not an assembly gap. See `../Pseudogenes/Asaccus_caudivolvulus_C9_lost/DETAILS.md`.
