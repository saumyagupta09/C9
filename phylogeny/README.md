# phylogeny/ — C9 gene/species tree
`C9_tree.treefile` / `.iqtree` — maximum-likelihood tree of the C9 sequences (IQ-TREE) with taxon list
(`C9_species_in_tree.txt`). Context tree for the gene family; the analysis-specific per-clade trees used
for RELAX/codeml are in `../gene_tree/`, and the all-copy tree for paralog-history in `../paralog_history/`.
